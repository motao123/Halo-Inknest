# Halo Theme InkNest

InkNest 是一款为 Halo 2 打造的内容型博客主题，主打大图 Banner、经典文章列表和沉浸式阅读体验。

## 特性

- 大图 Banner 与遮罩展示
- 接近 WordPress Dahuzi 风格的经典文章列表
- 文章页目录、阅读进度、代码复制和图片灯箱
- 移动菜单、返回顶部
- 分类、标签、归档、作者页适配
- 友情链接页，支持 Halo 链接插件
- 页脚社交链接、ICP 备案和公安联网备案展示
- 自定义 Head、CSS、JavaScript 和 Footer 代码

## 安装

1. 将 `Halo-InkNest` 放入 Halo 的 `themes` 目录。
2. 在 Halo 控制台启用 `InkNest`。
3. 在主题设置中配置 Logo、全局 Banner、菜单、文章、页脚和自定义代码。

## 兼容性

- Halo：`>= 2.25.0`
- 模板引擎：Thymeleaf
